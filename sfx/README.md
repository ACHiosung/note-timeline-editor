# sfx / 효과음 폴더

아래 파일을 이 폴더에 넣어 주세요.

| 파일명 | 용도 |
|--------|------|
| `tap.ogg` | 일반 노트 / 드래그 노트 탭음 |
| `tap_long.ogg` | 롱노트 시작(type 1) 탭음 |
| `tap_end.ogg` | 롱노트 끝(type -1) 탭음 |

* 파일이 없는 경우 해당 노트 타입의 효과음이 생략될 뿐, 재생에는 영향 없음.
* OGG 대신 WAV 나 MP3 를 사용하려면 `timeline-editor.html` 의 `SFX_FILES` 객체에서
  경로를 바꿔 주세요.
